# Module 2: User Management (Shaniya / Saravanan)

### Shaniya's Presentation Content (Frontend)
In Module 2, I focused on building the 'Entity Protocol'—our specialized interface for managing the human resources within our helpdesk. I developed the comprehensive 'Users' dashboard, which allows administrators to monitor, edit, and create user profiles with real-time feedback. One of my key contributions was the implementation of a dynamic, role-based user directory that leverages our 'DataTable' component for high-fidelity sorting and filtering. I also designed the 'Profile' management page, where users can personalize their accounts and view their professional activity in a clean, modern layout. To ensure operational efficiency, I integrated instant activation/deactivation toggles and role-assignment workflows that provide administrators with total control over system access. My work provides a powerful, intuitive management layer that simplifies complex administrative tasks while maintaining our project's commitment to a premium, interactive user experience.

### Saravanan's Presentation Content (Backend)
In Module 2, I engineered the robust 'User Management' API that serves as the backbone for our administrative operations. I developed a suite of high-performance routes that handle complex CRUD operations for user profiles, including sophisticated logic for password updates and role-based data isolation. A critical part of my work involved implementing the backend logic for administrative controls, such as the ability to instantly deactivate accounts and update departmental assignments without disrupting system integrity. I also designed the database relationships within our Prisma schema to ensure that user data is seamlessly linked across modules—from ticket assignments to SLA monitoring. My implementation provides the project with a secure and scalable repository for all user interactions, combining strict data validation with the flexibility needed to manage a diverse, multi-role user base effectively.
