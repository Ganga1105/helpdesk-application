# 🚀 Helpdesk Application (IBM Internship Project)

> A modular helpdesk system designed to streamline issue tracking, SLA management, and improve support team efficiency.

---

## 📌 Project Overview

This project is a scalable Helpdesk/Ticketing System inspired by Zoho Desk. It provides a centralized platform to manage customer queries, internal service requests, and support workflows efficiently.

---

## ✨ Key Features

- 🔐 Authentication & Role-Based Access Control  
- 👥 User Management System  
- 🎫 Ticket Lifecycle Management  
- 📎 File Attachments Handling  
- ⏱️ SLA Tracking & Escalation  
- 🔔 Real-Time Notification System  
- 📚 Knowledge Base Management  
- 📊 Reports & Analytics Dashboard  

---

## 🛠️ Tech Stack

- Frontend: React.js, Tailwind CSS  
- Backend: Node.js, Express.js  
- Database: MySQL / PostgreSQL  
- Authentication: JWT
